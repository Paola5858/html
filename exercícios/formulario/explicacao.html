Guia de Atributos do Formulário

Texto (text)

<input type="text" name="nome">
  • Usado para: Entrada de texto simples
  • Atributos: placeholder, required, maxlength

Senha (password)

<input type="password" name="senha">
  • Oculta os caracteres digitados
  • Atributos extras: minlength, maxlength, required

E-mail (email)

<input type="email" name="email">
  • Valida o formato de e-mail automaticamente
  • Atributos extras: required, placeholder

Número (number)

<input type="number" name="idade" min="0" max="120">
  • Entrada numérica com controle de mínimo e máximo

Rádio (radio)

<input type="radio" name="genero">
  • Permite selecionar uma opção entre várias
  • Usado para: gênero, escolhas únicas

Checkbox (checkbox)

<input type="checkbox" name="interesses">
  • Permite marcar várias opções
  • Usado para: interesses, preferências

Select (Lista suspensa)

<select name="cidade">
  <option>São Paulo</option>
</select>
  • Lista suspensa para escolher opções

Arquivo (file)

<input type="file" name="arquivo">
  • Permite upload de arquivos do dispositivo

Botão Submit

<button type="submit">Enviar</button>
  • Envia o formulário

Botão Reset

<button type="reset">Limpar</button>
  • Reseta todos os campos para vazio

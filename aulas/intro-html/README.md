# intro-html

primeiro contato com HTML. tags básicas, formatação de texto, listas e links.

## arquivo

- `teste.html` — primeiro arquivo HTML que escrevi

## tags praticadas

- `h1`, `h2` — hierarquia de títulos
- `p` — parágrafos
- `b`, `i`, `u`, `s` — formatação inline
- `a` — links
- `ul`, `ol`, `li` — listas ordenadas e desordenadas
- `blockquote` — citações
- `hr` — separador horizontal
- `<!-- -->` — comentários

## contexto

exercício de aula do SENAI. literalmente o dia 1 de HTML.

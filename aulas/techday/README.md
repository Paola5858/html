# techday

landing page de evento tech.

## arquivos

- `index.html` — estrutura HTML
- `styles.css` — estilos
- `hero.png`, `palestras.jpg`, `oficinas.jpg`, `convidados.jpg` — imagens

## estrutura

- hero section com CTA
- cards de conteúdo (palestras, oficinas, convidados)
- seção de inscrição
- footer

## técnicas aplicadas

- `background-image` no hero
- `box-shadow` em cards e botões
- `transition` no hover do botão
- layout centralizado com `max-width`

## contexto

exercício de landing page para evento fictício. foco em estrutura de conteúdo e hierarquia visual.

análise técnica do projeto

este projeto não é apenas uma tabela estilizada. é um exercício de design thinking aplicado a estrutura semântica, onde cada decisão técnica serve a um propósito conceitual e funcional.

table-responsive

garante que a tabela funcione em dispositivos móveis sem quebrar o layout. em telas pequenas, adiciona scroll horizontal mantendo a hierarquia visual intacta.

badges customizadas

uso badges do bootstrap como base semântica, mas sobrescrevo o estilo para criar identidade própria. cada eixo temático tem cor específica que reforça a categorização visual.

container centralizado

limita a largura máxima do conteúdo para manter legibilidade ideal. o olho humano processa melhor textos com 60-80 caracteres por linha.

glassmorphism

backdrop-filter cria profundidade sem peso visual. o efeito de vidro fosco adiciona sofisticação e separa camadas de informação sem usar bordas pesadas.

hierarquia tipográfica

uso clamp() para tipografia fluida que escala proporcionalmente. letter-spacing negativo em títulos grandes cria coesão visual, técnica comum em design editorial.

transições cubic-bezier

curvas de aceleração customizadas criam movimento mais natural. cubic-bezier(0.16, 1, 0.3, 1) produz efeito de "mola suave" que parece orgânico.

sobre a curadoria

escolhi essas obras porque todas elas operam em camadas. não são entretenimento passivo, são dissecações de sistemas — sejam eles políticos, psicológicos, morais ou sociais.

o que me interessa não é a história superficial, mas a estrutura invisível por trás dela. como linguagem molda pensamento. como prazer pode ser ferramenta de controle. como consciência individual entra em conflito com pressão coletiva.

este projeto também é sobre estrutura. não queria apenas listar livros e filmes. queria criar uma interface que refletisse o tema: algo que parece simples na superfície, mas tem camadas de decisão técnica e conceitual embaixo.

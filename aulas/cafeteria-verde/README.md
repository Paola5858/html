# cafeteria-verde

landing page de cafeteria com design moderno.

## estrutura

```
cafeteria-verde/
├── css/
│   └── style.css
├── imgs/
│   ├── logo-dot.png
│   ├── cup.png
│   ├── promo1.png
└── index.html
```

## componentes

- **topbar** — logo, navegação, CTAs (Entrar / Pedir agora)
- **hero** — título, descrição, botões, ilustração
- **banner** — cards promocionais (Latte doce de leite, Frappé de pistache)

## técnicas aplicadas

- navegação com flexbox
- hero com layout em duas colunas
- pills/badges para destacar novidades
- botões com variações (fill, outline)

## contexto

exercício de landing page comercial. foco em hierarquia de informação e CTAs.

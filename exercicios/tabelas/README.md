# tabelas

exercícios de listas e embed de vídeo com `iframe`.

## arquivos

- `silversprings.html` — página sobre Fleetwood Mac
- `webelong.html` — página sobre Mariah Carey
- `fleetwood.jpg`, `mariah.jpg` — backgrounds

## estrutura

- background fixo com `background-attachment: fixed`
- `backdrop-filter: blur()` — overlay fosco
- listas (`ul`, `ol`) — versos importantes, reflexões
- `iframe` — embed de vídeo do YouTube
- navegação entre páginas
- media queries — responsividade

## técnicas aplicadas

- `position: fixed` no background
- `backdrop-filter: blur(8px)` — efeito de vidro
- CSS inline no `<style>` (não em arquivo separado)
- `@import` do Google Fonts
- tabela semântica em `webelong.html` (thead, tbody)

## contexto

exercício de estrutura de conteúdo. tema: análise de músicas com listas, tabelas e vídeos.

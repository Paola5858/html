# site-evento2

variação de landing page de evento com galeria de imagens.

## arquivos

- `index.html` — estrutura HTML
- `styles.css` + `style.css` — dois arquivos CSS (provavelmente teste de organização)
- imagens: `evento.jpg`, `festa.jpg`, `festa1.jpg`, `festa2.jpg`, `festa3.jpg`, `brasil.jpg`, `love.jpg`, `party.jpg`, `party2.jpg`

## diferença do site-evento

- galeria de imagens (múltiplas fotos de festa)
- dois arquivos CSS separados
- mais imagens, menos texto

## contexto

exercício de variação. teste de organização de estilos em múltiplos arquivos.

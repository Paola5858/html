# site-evento

landing page temática de festa anos 2000.

## arquivos

- `index.html` — estrutura HTML
- `styles.css` — estilos com animações e gradientes
- imagens: `hero.jpg`, `fundo.jpg`, `playlist.jpg`, `outifit.jpg`, `convidados.jpg`, `eu.jpg`, `bombom.jpg`, `rodape.jpg`

## estrutura

- **hero** — título, descrição, CTA
- **navbar** — navegação por âncoras
- **cards** — playlist, outfit, convidados
- **seção organizadora** — foto pessoal com link pro Instagram
- **integração Spotify** — link pra playlist externa
- **seção bombom** — referência interna
- **footer** — créditos

## técnicas aplicadas

- navegação por âncoras (`#o-que-esperar`, `#confirmar`)
- `backdrop-filter: blur()` — efeito de vidro fosco
- `linear-gradient` com overlay em imagens
- `@keyframes` — animações customizadas
- `flexbox` — layout de cards
- `transition` + `transform` — hover interativo
- `box-shadow` com múltiplas camadas

## contexto

exercício livre. tema: festa anos 2000 com estética Y2K.

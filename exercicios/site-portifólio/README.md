# site-portifólio

página pessoal com tom informal.

## arquivos

- `index.html` — estrutura HTML
- `styles.css` — estilos
- `minha_foto.jpg`, `foto3.jpg` — galeria pessoal

## estrutura

- header com apresentação
- navegação (Início, Projetos, Contato)
- seção "Sabe quem eu sou?" — sobre mim
- lista de favoritos (filme, comida, lugar)
- galeria de fotos
- footer com créditos

## tom

informal, pessoal, com emojis e linguagem coloquial.

## contexto

exercício de apresentação pessoal com HTML/CSS puro. sem JavaScript, sem framework.
